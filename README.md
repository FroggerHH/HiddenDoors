# HiddenDoors

This mod just adds doors that look exactly like the walls.<br>
Localized to 31 language, thanks to Google Translate.

* Hideen Door Stone 2x2
* Hideen Door Stone 2x3
* Hideen Door Stone 4x2
* Hideen Door Wood
